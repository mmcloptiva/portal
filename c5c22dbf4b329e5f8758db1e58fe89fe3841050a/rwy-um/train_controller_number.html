Train Controller Number

Introduction

The train controller number identifies the train controller team.

Train controller number

The train controller number consists of 8 digits and is composed as follows:

Table 1. Train controller number
Digit Component
1 Call type = 7
2-6 UIN (User Identifier Number).
7-8 Function type as specified in the table below.

Function type

The function type is specified as follows:

Table 2. Train controller number function type
Call type digit 1

UIN

digit 2-6

Function code digit 7-8 Description
7 Train controller location 1 X X XX = Member number in train controller team
