Components of the Basic Network

Introduction

This section briefly describes some components of the basic mobile network.

Home Location Register

The Home Location Register (HLR) stores subscriber-related data, which include information stating whether the subscriber is a subscriber or not. The HLR transfers this subscription information to those network elements which need it to be able to provide the service.

Visitor Location Register

The Visitor Location Register (VLR) stores the subscriber data received from the HLR. The subscriber data include the Originating CAMEL Subscription Information (O-CSI) which is passed to the Mobile Switching Center (MSC) during the originating call processing. The VLR may also be requested for the subscriber‘s location or status by the HLR.

Mobile Switching Center

The purpose of the MSC is to set up calls from or towards the subscriber. The MSC may also be requested to monitor and report certain call events. Mobile Switching System (MSS) is a successor to Mobile Switching Center (MSC).
