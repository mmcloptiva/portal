EIRENE Numbering Plan

Introduction

This chapter describes the EIRENE numbering plan. It provides an overview of the different call types, the structure of the short dialing codes, and the structure of functional numbers, which can be divided into Call Type (CT), User Identifier Number (UIN) and Function Code (FC).

Call types

The following table shows all the possible call types, their meaning within the service and the number of digits that make up the functional number.

Table 1. Call types
Call Type Description Number of digits
0 No significance for the service
1 Short dialing codes 4
2 Train number 8 - 11
3 Engine number 11
4 Coach number 12
6 Maintenance and Shunting team number 10
7 Train controller number 8
8 MSISDN 20
9 Breakout numbers to another GSM-R network or to a private railway number.

3 (+ number of digits of PSTN or

GSM-R number, where applicable)
