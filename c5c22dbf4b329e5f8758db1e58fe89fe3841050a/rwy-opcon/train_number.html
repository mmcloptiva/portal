Train Number

Introduction

The train number identifies a train.

Train number

The train number consists of 8 to 11 digits and is composed as follows:

Table 1. Train number
Digit Component
1 Call type = 2
2-9 UIN (User Identifier Number). It is structured as follows:
0000X train number with one significant digit
000YX train number with two significant digits
00YXX train number with three significant digits
0YXXX train number with four significant digits
YXXXX train number with five significant digits
XXXXXX train number with six significant digits
XXXXXXX train number with seven significant digits
XXXXXXXX train number with eight significant digits
Y is not equal to 0.
10-11 Function type as specified in the table below.

Function type
