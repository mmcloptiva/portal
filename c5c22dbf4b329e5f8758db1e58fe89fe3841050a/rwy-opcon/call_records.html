Call Records

Introduction

Service events are monitored by SEP call records. One call record is generated for each call.

Ticket version

The ticket version for call records is 34.

CallTicketSwitch

The CallTicketSwitch for the call ticket is FCEBFFFFFFBFFFFF (as defined in the Profile entry in the related trigger information)

Note: For more detailed information on the standard format of call tickets, please refer to the product-specific statistics manuals.
