Structure of this Manual

This manual contains the following chapters:

  • Introduction to the Manual

    This chapter describes some basics for working with this manual.

  • Overview of the Service

    This chapter provides an overview of the service and its embedding in the system.

  • Service Description

    This chapter describes the features of the Optiva Railway service.

  • Structure of the Service

    This chapter describes the customer hierarchy.

  • Administration of Service Data

    This chapter provides information necessary for the correct administration of service data via the Web-based GUI and the batch interface.

  • Statistics

    This chapter provides information on the statistics tickets of the service.

  • EIRENE Numbering Plan

    This chapter provides an overview of the EIRENE numbering plan.
