Maintenance and Shunting Team Number

Introduction

The maintenance and shunting team number identifies the maintenance or the shunting team.

Maintenance team number

The team number consists of 10 digits and is composed as follows:

Table 1. Maintenance team number
Digit Component
1 Call type = 6
2-6 UIN (User Identifier Number).
7-10 Function type as specified in the table below.

Function type

The function type is specified as follows:

Table 2. Specified maintenance team number function type
Call type digit 1 UIN digit 2-6 Function code digit 7-10 Description
6 Team location TT Y X X See table bellow
Table 3. Maintenance team number function type
Function type Component
TT = 1 ... 4 reserved for international use
TT = 5

Shunting team

y =

0 - Team leader

1 ... 3 - Team member

4 - Train driver

5 - Device for connection control reserved for international use

6 ... 9 - reserved for international use

xx =

00 - reserved

01 ... 20 - Shunting groups

21 ... 99 - reserved
TT = 6 ... 9 reserved for national use
TT = 0 reserved for international use
