Functional Numbers

Introduction

This section describes the functional numbers as defined by the EIRENE numbering plan.

Functional numbers

Functional numbers consist of

  • a call type prefix
  • a User Identifier Number, which uniquely identifies a train, an engine, etc.
  • a function code which defines a function within a group.

Two types of functional numbers can be distinguished:

  • national EIRENE numbers (which do not contain a Railway Access Code (RAC))
  • international EIRENE numbers (which contain a Railway Access Code)
