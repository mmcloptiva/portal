Short Dialing Codes

Introduction

Short dialing codes enable speedy, uncomplicated contact between the train driver and the controller responsible for the respective section. Instead of a long train number, the train driver must only dial a four-digit functional number. Three short dialing codes whose meaning can be found in the following table are used by the GSMR service.

Short (dialing) code

A short dialing code consists of 4 digits and is designated as call type 1. The following table shows the structure and meaning of the short dialing codes.

Table 1. Short (dialing) code
Call type digit 1 Digit 2-4 Description
1 2 X X * Call from cab driver to primary controller
3 X X * Call from cab driver to secondary controller
4 X X * Call from cab driver to electricity controller
5 X X

Route to most appropriate ERTMS (European Rail

Traffic Management System) / ETCS (European Train

Control System) RBC (Radio Block Center)
1 6 1 2 High priority call confirmations including railway emergency calls. This short code is not handled by the GSM-R.

*) Digits 3 and 4 (XX) may be used to provide supplementary location information within a cell. If no additional information is available, the default value is 00.

Special short code

One special short code is supported as well:

Table 2. Special short code
Code Description
112 European emergency number
